# Python---Mundo-2
Python - World 2 (Curso em Vídeo)
This repository contains the exercises and challenges proposed during World 2 (Mundo 2) of the Python 3 course, taught by Professor Gustavo Guanabara from the Curso em Vídeo platform.

About the Project
"World 2" of the Python 3 course focuses on repetition and control structures, deepening the knowledge acquired in World 1.

The purpose of this repository is to serve as a record and study material for the proposed challenges, covering topics such as:

Repetition structures (for and while).

Nested control structures.

Stopping loops (break).
